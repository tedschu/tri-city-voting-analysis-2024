# 2024 Presidential Election Data Analysis - Kane County, Illinois

Analysis of voting patterns by precinct and town in Kane County, IL - specifically for St. Charles, Geneva, and Batavia.

## Overview

This project analyzes Democratic and Republican voting patterns across precincts three Kane County towns. The analysis identifies top-performing precincts for each party and visualizes voting distributions.

## Data

- **Source**: https://electionresults.kanecountyil.gov/2024-11-05/Precincts/
- **Scope**: Kane County precincts
- **Fields**: Town, Precinct, Democratic votes, Republican votes, Total votes

## Analysis Includes

- Vote totals by town
- Party vote percentages by precinct
- Top 5 precincts by Democratic vote percentage
- Top 5 precincts by Republican vote percentage
- Comparative visualizations

## Technologies Used

- Python 3.x
- pandas - data manipulation
- seaborn/matplotlib - visualization
- Jupyter Notebook

## Files

- `analysis.ipynb` - Main analysis notebook
- `data/election_data.csv` - Raw election data
- `README.md` - This file
